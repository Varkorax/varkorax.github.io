Milky Way Bar Crawl - Setup

Preparing the Games

▾

Before we can start modding, let's make sure everything's in place.

Installing the Game

Before anything else, let's make sure your Mass Effect Legendary Edition installation is up and running.

!

If you’ve tried modding before, I recommend starting fresh. Locate your current installation, delete the entire game folder, and redownload the game.
Annoying, I know. But a clean slate will save you from a lot of trouble down the line.


When (re)installing, choose a location outside protected Windows directories (like Program Files) to avoid potential permission issues.

If you are using the EA App, simply pick a custom install path - something like: C:\Games\EA

If you are on Steam, the ideal setup is a second library on a separate SSD.
If you don’t have a second drive or SSD, you can use the Steam Library Setup Tool to create a second library on your main drive (e.g., C:\Games\Steam).

First Boot


Optional: Before starting the game you may wish to disable the in-game overlay, since there is a small chance it can interfere with modded launches.
(If you installed via Steam, you will want to disable both the Steam and EA App overlays.)

To disable the Steam Overlay:

  1. Locate Mass Effect Legendary Edition in your Steam Library.
  2. Right-click it → Properties.
  3. Under the General tab, uncheck Enable the Steam Overlay while in-game.

To disable the EA App Overlay:

  1. Click the three lines in the top-left corner of the EA App.
  2. Go to Settings → Application.
  3. Scroll to In-game Overlay and uncheck it.

Optional: You may also want to disable automatic updates, though the chance of the LE getting another update are virtually zero.

To disable automatic updates in the EA App:

  1. Click the three lines in the top-left corner of the EA App.
  2. Go to Settings → Application.
  3. Scroll to Automatic Updates and uncheck it.


Once everything is in order, start each game once and make any graphical adjustments you need.

Your game(s) should now be ready, which means we can move on to:

Preparing the Mod Manager

▸

The Mass Effect community has developed its own dedicated modding tools, and those are what we’ll be using here.
Vortex, MO2, or similar tools are not supported.

Installing M3 (ME3Tweaks Mod Manager)

M3 is the central hub for modding the Legendary Edition. It manages and installs mods across all three games.
You can download the latest version from the official GitHub page: ME3Tweaks Mod Manager Releases

!

Do not extract M3 inside your game directory or a protected system folder (e.g. Program Files).
I recommend keeping all modding related files in a separate, easy-to-access location - for example: C:\Mods\MELE

Optional: Backup

As mentioned earlier, if you have the space available I recommend making a backup of your unmodded game.
This allows you to restore a clean copy in case something breaks later.
M3 will remind you about this each time it starts, but you can safely close that window if you don’t plan to make one.

If you do wish to create one, click on Backup & Restore → Backup, follow the on-screen instructions in M3 and select an appropriate location (e.g. D:\Backup\MELE).



M3 should now be ready, which means there is only one last thing to take care of before you can start modding:

Installing Mods

▸

We’re almost ready now. All that remains is one final step: learning how to install mods.
You have two options:

Downloading Through M3

M3 can download mods directly from Nexus Mods:

  1. Click the Nexus icon in the top-right corner and log into your Nexus account.
  2. Once authorized, you should now see your name in the top right. Click on it and in the now open window click on Configure 'Download with Manager' and then Register ME3Tweaks Mod Manager as handler.
  3. Now, when you click Download with Mod Manager on a Nexus page, M3 will handle it automatically.

Downloading Manually

Some mods must be downloaded manually. These usually come as .zip or .7z archives.

Keep them organized near your M3 folder - for example:
C:\Mods\MELE\ME1 Downloads

Once downloaded, you can:

  • Drag and drop the archive into M3, or
  • Use Mod Management → Import Mod → Archive.



No matter how you get them you'll see an import window which will show details about the mod and give you the options to install it immediately or just add it to your library.

Always import your mods. It makes managing and troubleshooting much easier later.

?!

Do not try to manually install mods.
M3 performs a lot of technical wizardry while installing them, so just dragging and dropping the files into the game folder will do nothing at best, and break your game at worst.


You can now click on the mod in the left panel to select it, and then click Apply Mod in the bottom right to install it.
You may be asked to select patches or customize something. If you are unsure about something I recommend reading through the Nexus page of the mod.

Optional: Uninstalling Mods

Should you need to uninstall a mod you can do so like this:

  1. Select the game in the dropdown menu.
  2. Click on Manage Target.
  3. Find the mod you wish to uninstall.
  4. Click Disable to only remove it temporarily or Delete to remove it entirely.


Load Order

A quick note about load order:
Don't worry about it!

There are only two golden rules you need to know before diving in:

1. Content first, Textures last!
Due to technical reasons outside the scope of this guide, textures should always be installed last. Once you begin texture modding, you cannot install any new content mods or updates. You’ll see plenty of warnings about this, but trust me - doing so will break your game.

2. Read the Nexus page, check for compatibility!
It’s the best way to understand what a mod does, whether it needs a patch, or if it conflicts with another.

Failure to follow these rules will be met with swift reprimand - and the knowledge that somewhere I’m shaking my head at you in quiet disapproval.



Everything should be ready now, which means we can finally get to the fun part: selecting mods!

Choosing Your Route

▸

Before diving into the actual mod list, there’s one last decision to make: Batch or Manual installation. This choice will determine how you proceed through the guide.

Option A: Batch

This uses a preconfigured setup file for M3. You’ll still download each mod manually from within M3, but installation and configuration happen automatically.

  • Pro: Extremely convenient - M3 takes care of setup and settings for you. Just confirm downloads and click install.
  • Con: Inflexible - You get my intended setup, nothing more. Editing batch files is possible but best left for when you’re more experienced.

Option B: Manual

This method means installing each mod yourself from one of the lists.

  • Pro: Full control - pick exactly what you want. Don’t like one of my choices? Skip it.
  • Con: More room for mistakes - blindly adding mods can easily cause issues if you’re not careful.

Which Should I Pick?

The choice, in my opinion, depends on your experience level:

  • Novice: If you’re new to modding or want a curated, hassle-free experience, use the Batch setup.
  • Veteran: If you’re familiar with modding or want to customize everything yourself, go Manual.

With that decision made, we can move on to a quick stop before the main event:

Aperitif: Launcher

▸

Before diving into the main games, there’s one small area we can polish first – the Launcher itself.
This section is optional but quick, and helps tidy up the interface you’ll be seeing every time you boot up.

Remember: Download through M3 or manually → Import the mod → Click Apply Mod.
If you haven't made a backup, M3 will warn you about that, but you can safely continue.

Myriad Pro Begone

Replaces the font used throughout the launcher with one that better fits the series’ visual identity.
Note: The downloaded archive includes versions for each game and the launcher – you can import them all or just install the launcher version for now.

Smoother GUI (Launcher)

Increases the fps of the launcher animations/GUI elements.
Note: I recommend the "Alternative" option.



That’s it for the launcher! These tweaks are small but set a nice tone for what’s to come.
With the setup complete and your preferred route chosen, it’s time for the main course: LE1 Mods (Or, you know, LE2 or LE3).
