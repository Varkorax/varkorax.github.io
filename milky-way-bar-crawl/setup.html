Milky Way Bar Crawl - Setup

Preparing the Games

▾

Before we can start modding there are a few things we need to take care of, starting with:

Installation

First, we’ll make sure your copy of Mass Effect Legendary Edition is installed and working properly.

!

If you’ve tried modding before, I strongly recommend starting fresh.
Locate your current installation, delete the entire game folder, and redownload the game.
Annoying, I know. But a clean slate will save you from a lot of trouble later.


When (re)installing, choose a location outside protected Windows directories (like Program Files) to avoid permission issues.

If you are using the EA App, simply pick a custom install path (e.g. C:\Games\EA).

If you are on Steam, the ideal setup is a second library on a separate SSD.
If you don’t have a second SSD, you can use the Steam Library Setup Tool to create a second library on your main drive (e.g. C:\Games\Steam).

First Boot


Optional: Before starting the game, you may wish to disable the in-game overlay, as it can occasionally interfere with modded launches.
(If you installed via Steam, you’ll want to disable both the Steam and EA App overlays.)

To disable the Steam Overlay:

  1. Locate Mass Effect Legendary Edition in your Steam Library.
  2. Right-click it → Properties.
  3. Under the General tab, uncheck Enable the Steam Overlay while in-game.

To disable the EA App Overlay:

  1. Click the three lines in the top-left corner of the EA App.
  2. Go to Settings → Application.
  3. Scroll to In-game Overlay and uncheck it.

Optional: You may also want to disable automatic updates, though the chance of LE getting another update is virtually zero.

To disable automatic updates in the EA App:

  1. Click the three lines in the top-left corner of the EA App.
  2. Go to Settings → Application.
  3. Scroll to Automatic Updates and uncheck it.


Once everything is in order, start each game once and make any graphical adjustments you need.
In the launcher, select options and make sure subtitle size is set to small.

Your games should now be ready, which means we can move on to the tool that will actually handle your mods:

Preparing the Mod Manager

▸

The Mass Effect community has developed their own modding tools, and those are what we’ll be using here.
Vortex or MO2 will not work and are not supported.

Installing M3 (ME3Tweaks Mod Manager)

M3 is our central hub for modding Mass Effect. It manages and installs mods across all three games, OT and LE.
You can download the latest version from the official GitHub page: ME3Tweaks Mod Manager Releases

!

Do not extract M3 inside your game directory or a protected system folder.
I recommend keeping all modding related files in a separate, easy-to-access location (e.g. C:\Mods\MELE).

Optional: Backup

As mentioned earlier, if you have the space available I recommend making a backup of your unmodded game.
This allows you to restore a clean copy in case something breaks later.
M3 will remind you about this each time it starts, but you can safely close that window if you don’t plan to make one.

If you do wish to create one, click on Backup & Restore → Backup, follow the on-screen instructions in M3 and select an appropriate location (e.g. D:\Backup\MELE).



M3 should now be ready, which means there is only one last thing to take care of before you can start modding:

Installing Mods

▸

Ok, I lied. But we’re almost ready now. First though you need to learn how to install mods.
You have two options:

Downloading Through M3

M3 can download mods directly from Nexus Mods:

  1. Click the Nexus icon in the top-right corner and log into your Nexus account.
  2. Once authorized, you should now see your name in the top right. Click on it and, in the window that opens, click Configure "Download with Manager", then Register ME3Tweaks Mod Manager as handler.
  3. Now, when you click Download with Mod Manager on a Nexus page, M3 will handle it automatically.

Downloading Manually

You may prefer to (or in some cases need to) download something manually. These usually come as .zip or .7z archives.

Keep them organized near your M3 folder (e.g. C:\Mods\MELE\ME1 Downloads).

Once downloaded, you can:

  • drag and drop the archive into M3, or
  • use Mod Management → Import Mod → Archive.



No matter how you get them, you’ll see an import window that shows details about the mod and gives you the option to install it immediately or just add it to your library.

Always import your mods. It makes managing and troubleshooting much easier later.

?!

Do not try to manually install mods.
M3 performs a lot of technical wizardry while installing them, so just dragging and dropping files into the game folder will do nothing at best and may break your game at worst.


You can now click on a mod in the left panel to select it, and then click Apply Mod in the bottom right to install it.
You may be asked to select patches or customize something. If you are unsure about a choice, I recommend reading through the mod’s Nexus page.

Optional: Removing Mods

Should you wish or need to uninstall or disable a mod, you can do so like this:

  1. Select the right game from the center dropdown menu.
  2. Click on Manage Target in the upper right. This will show a list of all installed mods.
  3. Find the mod you wish to uninstall or disable.
  4. Click Disable to remove it temporarily, or Delete to remove it entirely.

Golden Rules

A quick note about load order: Don’t worry about it!

There are only two golden rules you need to know before diving in:

1. Content first, Textures last!
Due to technical reasons outside the scope of this guide, textures should always be installed last. Once you begin texture modding, you cannot install new content mods or updates. You’ll see plenty of warnings about this, but trust me, doing so will break your game.

2. Read the Nexus page, check for compatibility!
It’s the best way to understand what a mod does, whether it needs a patch, or if it conflicts with another.

Failure to follow these rules will be met with swift reprimand, and the knowledge that somewhere I’m shaking my head at you in quiet disapproval.



With all this newfound knowledge under your belt only one choice remains before you can start modding:

Choosing Your Route

▸

Generally there are two ways you can proceed through the main portion of this guide:
Batch or Manual

Option A: Batch

I provide you with a preconfigured setup file for M3. You’ll still have to download each mod from within M3, but installation and, more importantly, configuration happen automatically.

  • Pro: Convenient. M3 takes care of setup and settings for you. Just confirm downloads and click install.
  • Con: Inflexible: you get my intended setup, nothing more. Editing batch files is possible but best left for when you’re more experienced.

Option B: Manual

Downloading and installing each mod yourself using one of the provided lists.

  • Pro: Full control. You pick exactly what you want. Don’t like one of my choices? Skip it.
  • Con: More room for mistakes if you blindly add or skip things without checking.

Which Should I Pick?

In this barman's opinion, the choice mostly comes down to your comfort level:

  • Curated: If you’re new to modding or want a curated, hassle-free experience, use the Batch setup.
  • Trailblazer: If you’re familiar with modding or want to customize everything yourself, go Manual.

With that decision made, we can have a little taste of what modding the games will be like with:

Aperitif: Launcher

▸

Before diving into the main games, there’s one small area we can polish first: the Launcher itself.
Technically this section is optional, but it's quick, helps tidy up an interface you’ll be seeing a lot, and is a good low stakes test run.

(Pssst! Remember: Download through M3 or manually → Import the mod → Click Apply Mod.)
If you haven't made a backup, M3 will warn you about that every time you install a mod, but you can safely continue.

Myriad Pro Begone

Replaces the font used throughout the launcher with one that better fits the series’ visual identity.
Note: The downloaded archive includes versions for each game and the launcher. You can import them all or just install the launcher version for now.

Smoother GUI (Launcher)

Increases the fps of the launcher animations and GUI elements.
Note: I recommend the "Alternative" option.



And that's it for the launcher. These tweaks are small but set a nice tone for what’s to come.
With the setup complete and your preferred route chosen, it’s time for the main course!
